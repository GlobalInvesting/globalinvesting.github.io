# Global Investing — Forex Fundamental Analysis Dashboard

A quantitative fundamental analysis platform for the 8 major G8 currencies. Combines macroeconomic data, institutional positioning, and AI-generated narratives into a single unified strength score per currency.

**[globalinvesting.github.io](https://globalinvesting.github.io/)** · [English version](https://globalinvesting.github.io/en.html)

---

## Features

- **Fundamental strength score (0–100)** per currency — quantitative ranking of all 8 G8 currencies updated daily
- **Interactive indicator heat map** — 22 macroeconomic indicators across all currencies in a single view
- **LONG/SHORT pair signals** — automatically generated with confidence levels based on score differentials
- **Carry trade analysis** — live interest rate differential rankings with risk regime indicator
- **Institutional COT positioning** — CFTC Leveraged Funds data with contrarian signal logic
- **Economic calendar** — live events with actuals vs forecasts and automatic timezone conversion
- **AI-written narratives** — daily fundamental analysis per currency
- **RSS feeds** — subscribe to news, AI analysis, and G8 scores
