About

Global Investing FX Terminal

A professional-grade foreign exchange monitoring platform built entirely on publicly available data feeds, institutional positioning reports, and AI-assisted narrative synthesis. Free to use. No login required. No data sold.

What it is

The FX Terminal is a single-page dashboard designed for serious FX monitoring. It consolidates real-time price feeds, central bank policy data, CFTC institutional positioning, US Treasury yield curve metrics, cross-asset correlation data, and AI-generated market regime narrative — all in one view, updating continuously during market hours.

It was built to answer a specific problem: professional-quality FX context is fragmented across Bloomberg terminals, Reuters Eikon, broker platforms, and government data portals. This terminal aggregates the most signal-rich public data into a coherent, fast, and always-available interface.

Architecture

The terminal is a static site deployed on GitHub Pages. There is no backend server, no database, and no runtime cost. Two GitHub repositories work together:

engine (private) Python scripts and GitHub Actions workflows that fetch, process, and write JSON data files to the public repo on schedule. Contains AI prompts and scoring logic.
site (public) Static HTML/CSS/JS terminal UI + JSON data files. GitHub Pages serves it globally. The frontend reads data files with fetch() calls — no API keys exposed.
TradingView Embedded widgets for live charting, forex heatmap, economic calendar, ticker tape, and economic map. No API key required for public embeds.

This architecture means the terminal is available 24/7, loads fast (no server round trips for most data), and can sustain indefinitely at zero marginal cost.

AI usage — intentionally limited

AI (Groq / llama-3.3-70b-versatile) is used for exactly two outputs:

Narrative bar
A 2–3 sentence market summary updated 3× daily (06:30, 12:30, 21:30 UTC), plus a regime label: RISK-ON / RISK-OFF / MIXED / NEUTRAL. Synthesizes rates, sentiment, cross-asset flows, and macro events into a single readable context.
Alerts & Market Signals
4–6 prioritized, actionable market signals published alongside the narrative. Each signal includes an asset, a direction, a rationale, and a confidence level based on cross-referencing rate differentials, COT positioning, and recent price action.

All other sections — News Feed, Quote Bar, COT, Rates, Heatmap, Charts — consume real data directly without AI filtering or summarization. AI does not generate or alter price data.

Currencies covered

USD · EUR · GBP · JPY · AUD · CAD · CHF · NZD — the eight G8 major currencies, covering over 85% of global daily FX turnover by BIS estimates.
